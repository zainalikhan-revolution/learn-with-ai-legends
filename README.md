
# 🏛️ Learn with AI Legends
### Learn from Einstein. Think like Curie. Built with Claude Opus 4.7.
#### Anthropic Built with Opus 4.7 Global Hackathon — April 2026

An AI-powered video tutoring platform where students learn directly from history's greatest minds — through personalized, interactive, curriculum-aligned sessions powered by Claude Opus 4.7.

---

## 🌍 The Problem

Millions of students around the world prepare for critical exams with no access to quality teachers. Private tutors are expensive. Schools are overcrowded. Students in developing regions are left with textbooks and no one to explain them.

**Learn with AI Legends is built to change that.**

---

## 💡 What It Does

Students don't just chat with an AI. They sit down with **Albert Einstein** for Physics, **Marie Curie** for Chemistry, or **Allama Iqbal** for Literature — and get a fully personalized, one-on-one tutoring session aligned to their exact syllabus.

---

## ✨ Core Features

### 🧑‍🔬 AI Legend Tutors
Choose your tutor — Einstein, Curie, Iqbal, and more. Each legend has a unique teaching personality, communication style, and area of expertise. The AI adapts its tone, depth, and examples based on the student's level.

### 🔷 Interactive 3D Visualizations
Physics and Chemistry concepts come alive through real-time 3D models generated directly in the platform. Concave mirrors, focal points, pendulums, molecular structures — students don't just read about them, they see and interact with them.

### 🕸️ AI Mind Map Generator
One click generates a fully interactive concept map connecting related ideas — from Simple Harmonic Motion to Pendulums to Wave Equations. Students explore at their own pace, with dynamic graphs and animations at every node.

### 📷 Live Vision Mode *(Most Powerful Feature)*
The AI tutor can **see the student through their camera** in real time. It analyzes the student's environment, recognizes objects, and makes the session completely immersive and context-aware. Ask Einstein what he sees — and he'll tell you.

### 💬 Text Chat Mode
A dedicated chat interface for quick, comprehensive answers to foundational questions — available 24/7, infinitely patient, always curriculum-aligned.

## 🚀 Getting Started

### Prerequisites
- Python 3.9+
- Anthropic API Key

### Installation

```bash
git clone https://github.com/YOUR-USERNAME/learn-with-ai-legends
cd learn-with-ai-legends
pip install -r requirements.txt
```

### Setup

```bash
export ANTHROPIC_API_KEY=your_key_here
```

### Run

```bash
python main.py
```
